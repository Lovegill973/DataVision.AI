# DataVision AI

DataVision AI is an AI-powered Smart Data Analyzer designed to simplify data exploration and visualization. Users can upload CSV or Excel datasets, generate interactive charts, analyze trends, receive AI-generated insights, and export reports in PDF format.

## Features

- Upload CSV and Excel datasets
- Interactive data visualizations
- KPI dashboard and summary statistics
- Correlation heatmaps
- AI-powered insights using Gemini AI
- PDF report generation
- User-friendly Streamlit interface

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Matplotlib
- Google Gemini AI
- FPDF2
